drawiterm is a terminal-native clone of Excalidraw — a standalone TUI drawing tool for system design sketches.

**Why:** Engineers who work primarily in terminals (especially over SSH) have no good option for quick, informal system design diagrams. Excalidraw is the gold standard for this style of sketching but requires a browser. drawiterm brings that experience to the terminal.

**How to apply:** All design decisions should optimize for the "quick system design sketch" use case — not pixel-perfect art, not ASCII art generators, but Excalidraw-style informal boxes/arrows/labels diagrams that communicate architecture.
